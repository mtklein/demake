# Session tapes

Raw takes from the music sessions, exactly as each agent-musician returned
them. Everything here has already been pressed into `tools/music/songs.py`
(the canonical source); these are kept as provenance and as remix stock.

- `bass` `drums` `tpt` `sax` — the four Kind of Azure players, who recorded
  their parts without hearing each other (the Kind of Blue method). Validated
  and spliced by `assemble.py`.
- `gaia` `selune` `tadpole` `bossa` — the single-composer commissions
  (Holst movement, Eyes-on-Me ballad, surf, bossa nova), pressed by `press.py`.
- `disco` `soad` `prelude` — the re-cuts (house battle theme, nu-metal boss,
  title-duet enrichment), applied in place by `recut.py`.
- `battle` — an outtake: the orchestral Draw Steel! re-cut that was finished
  and validated, then superseded the same hour by the disco reconception.
